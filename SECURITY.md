# Security Policy

## Supported Versions
This project currently supports the latest `main` branch. If a release process is added, this policy will be updated.

## Reporting a Vulnerability
Please report security issues via GitHub Security Advisories for this repository.

If you cannot use the advisory flow, open a new GitHub issue with minimal details and request a private channel.

## Disclosure Process
- We will acknowledge receipt as soon as possible.
- We will investigate and provide a remediation plan or timeline.
- Once fixed, we will coordinate on a public disclosure if appropriate.

## Third-Party Dependencies
This repo vendors LuaSocket/LuaSec under `vendor/`. If a security issue affects
bundled dependencies, we will assess impact and update the vendored code or
document mitigations. Please report dependency issues through the same security
channels.
